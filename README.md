# Verificador de Alistamento Militar

Projeto de estudo desenvolvido em Python que calcula a idade do usuário com base no ano de nascimento e verifica sua situação em relação ao alistamento militar obrigatório no Brasil.

## Objetivo do projeto
Praticar conceitos fundamentais da linguagem Python, como:
- Entrada de dados pelo terminal
- Operações aritméticas
- Estruturas condicionais (`if`, `elif`, `else`)
- Uso da biblioteca `datetime`

## Funcionalidades
- Solicita o ano de nascimento do usuário
- Calcula automaticamente a idade atual
- Informa a situação do usuário em relação ao alistamento militar:
  - Ainda não precisa se alistar
  - Deve se alistar no ano atual
  - Está em atraso
  - Já passou do período obrigatório

## Tecnologias utilizadas
- Python 3
- Biblioteca padrão `datetime`

## Como executar
1. Certifique-se de que o Python esteja instalado no computador.
2. Baixe o arquivo `verificador-alistamento-militar.py` do projeto.
3. Execute o arquivo pelo terminal ou por uma IDE de sua preferência, como:
   - PyCharm
   - Visual Studio Code
   - IDLE
  
## Exemplo de uso
Entrada:
Digite o ano de nascimento: 2005

Saída:
Você tem 21 anos.
Situação: deve se alistar este ano.
